Model Architecture

Input (32x32x3 — CIFAR-10 images) | Prep Layer Conv2d(3-64, 3x3) > BN > ReLU [RF: 3x3] | Layer 1 Conv2d(64-128) > MaxPool > BN > ReLU [RF: 11x11] + ResidualBlock(128-128) | Layer 2 Conv2d(128-256) > MaxPool > BN > ReLU [RF: 27x27] | Layer 3 Conv2d(256-512) > MaxPool > BN > ReLU [RF: 59x59] + ResidualBlock(512-512) | Global Pool MaxPool(4x4) [RF: 88x88] | Output Linear(512-10 classes)

Key Techniques

1

Custom Residual Blocks with Skip Connections

Enables gradient flow through deeper architectures, prevents vanishing gradients. Receptive fields calculated from 3x3 to 88x88.

2

Advanced Albumentations Augmentation Pipeline

RandomCrop, HorizontalFlip, and CutOut (CoarseDropout 16x16) filled with dataset mean. Significantly improves model generalization.

3

One Cycle Policy — Super Convergence

Linear annealing with dynamic momentum adjustment (0.95 to 0.85 to 0.95). Peak LR at 30% of training. Achieves 40% faster convergence.
